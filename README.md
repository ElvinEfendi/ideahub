## Get Started

1. Install bundle
  
  bundle install

2. Please use passenger as rails server

  gem install passenger

2. Add /etc/hosts entry

  127.0.0.1 ideahub.local

3. Start server using your own username

  rvmsudo passenger start -p 80 -a ideahub.local --user sandip
